## Terms
Assume A is control, B is test.

```ad-sam
Dependent Variable
- Continuous
- Proportions
```

```ad-sam
Are A & B 2 separate, independent groups?
- Yes --> **Unpaired**
- No --> **Paired** (same group, multiple periods)
```

```ad-sam
Are we simply comparing B to a known benchmark?
- Yes --> **One-Sample**
- No --> **Two-Sample**
```

```ad-sam
Does B need to out-perform control?
- Yes --> **One-Sided**
- No --> **Two-Sided**
```
