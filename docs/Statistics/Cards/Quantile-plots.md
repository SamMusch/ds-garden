- **1D**: Compares **distributions** (single-variable). [Image](https://i.imgur.com/ALHW7ej.png)

- **2D**: Visualizes **relationships** (joint-variable focus). [image](https://i.imgur.com/0KMGiD1.png)

| **Aspect**     | **1D Quantile Plot (Q-Q Plot)**                                                                       | **2D Quantile Plot**                                                                                          |
| -------------- | ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Definition** | Compares the quantiles of a dataset to another distribution.                                          | Visualizes the joint distribution of two variables using quantiles.                                           |
| **Purpose**    | Identify deviations like skewness or outliers.<br><br>Distribution analysis (e.g., normality checks). | Examine density patterns or dependencies.<br><br>Joint distribution analysis (e.g., correlation, clustering). |
| **Example**    | Comparing data v normal distribution                                                                  | Joint distribution for height and weight                                                                      |
