### Cloud Computing

- [AWS](https://www.oreilly.com/search/skills/amazon-web-services-aws/): a cloud infrastructure platform offered by Amazon. It provides scalable compute, storage, networking, and managed services for hosting applications and data systems.

- [Cloud Platforms](https://www.oreilly.com/search/skills/cloud-platforms/): provide on-demand infrastructure and managed services for computing, storage, networking, and application deployment across distributed data centers.

### Data Engineering

- [Data Warehouse](https://www.oreilly.com/search/skills/data-warehouse/): a centralized system optimized for analytical queries, reporting, and historical data analysis across structured datasets.

- [Apache Spark](https://www.oreilly.com/search/skills/apache-spark/): a distributed data processing engine designed for large-scale batch and streaming workloads.

- [Kafka](https://www.oreilly.com/search/skills/kafka/): a distributed event streaming platform used for real-time data pipelines and message-driven architectures.

- [Data Lake](https://www.oreilly.com/search/skills/data-lake/): a centralized storage system that holds raw, structured, semi-structured, and unstructured data at scale.

- [Streaming & Messaging systems](https://www.oreilly.com/search/skills/streaming-messaging/): enable real-time data flow between producers and consumers with low latency.

- [NoSQL Databases](https://www.oreilly.com/search/skills/nosql-databases/): store non-relational data and are optimized for scalability, flexibility, and high-throughput workloads.

### AI & ML

- [Generative AI](https://www.oreilly.com/search/skills/generative-ai/): focuses on models that create new content such as text, images, code, or audio based on learned patterns.

- [Machine Learning](https://www.oreilly.com/search/skills/machine-learning/): enables systems to learn patterns from data and make predictions without explicit programming.

- [Artificial Intelligence (AI)](https://www.oreilly.com/search/skills/artificial-intelligence-ai/): encompasses systems that perform tasks requiring reasoning, perception, or decision-making.

- [Deep Learning](https://www.oreilly.com/search/skills/deep-learning/): uses multi-layer neural networks to model complex patterns in large datasets.

- [Reinforcement Learning](https://www.oreilly.com/search/skills/reinforcement-learning/): trains agents to make sequential decisions through reward-based feedback.

- [NLP](https://www.oreilly.com/search/skills/natural-language-processing/): enables computers to understand, interpret, and generate human language.

- [MLOps](https://www.oreilly.com/search/skills/mlops/): focuses on deploying, monitoring, and maintaining machine learning models in production environments.

### Software Architecture

- [Object-Oriented](https://www.oreilly.com/search/skills/object-oriented-architecture/): an architecture that organizes software around objects that encapsulate data and behavior.

- [Distributed Systems](https://www.oreilly.com/search/skills/distributed-systems/): consist of multiple networked components that coordinate to achieve a common goal.

- [Domain-Driven Design](https://www.oreilly.com/search/skills/domain-driven-design/): structures software around core business concepts and domain logic.

- [Architectural Patterns](https://www.oreilly.com/search/skills/architectural-patterns/): provide reusable solutions to common system design problems.

### IT / Ops

- [Kubernetes](https://www.oreilly.com/search/topics/kubernetes/): a container orchestration platform for deploying, scaling, and managing containerized applications.

- [Docker](https://www.oreilly.com/search/topics/docker/): enables applications to be packaged and run in isolated, portable containers.

- [GitHub](https://www.oreilly.com/search/topics/github/): a platform for source control, collaboration, and software development workflows.

- [Computer Networking](https://www.oreilly.com/search/topics/computer-networking/): the design and operation of systems that transmit data between computers and devices.
